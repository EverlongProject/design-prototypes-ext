# manulife-demo-april

Product design prototype. Mobile PWA exploring a conversational benefits experience.

## Stack

- Vite + React 18 + Tailwind CSS
- Framer Motion for transitions
- `vite-plugin-pwa` for installable standalone mode

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Published at `/manulife-demo-april/` via GitHub Pages (see repo root workflow).
