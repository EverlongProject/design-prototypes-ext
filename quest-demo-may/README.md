# quest-demo-may

Desktop web prototype of the Quest AI Care Navigation demo for the May 7, 2026 Quest onsite. Mirrors the structure and conventions of `highmark-demo-may/` in this repo.

Start with `SPEC.md` for the project brief, `DEMO_SCRIPT.md` for the beat-by-beat conversation script, and `tailwind.config.js` for the (placeholder) design tokens.

```bash
npm install
npm run dev    # http://localhost:5175
```
